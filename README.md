## 專案說明

> 使用 React + Vite 建構，支援 RWD，部署於 GitHub Pages。

---

## 作品發想由來

這個作品源自培訓期間的作業題目，作業要求是練習完整的購物車流程，以及使用 Redux Toolkit 實作全域提示訊息與購物車 icon 上的數量顯示。畫面不在評分範圍內，但在開發時主動進行介面設計與視覺優化，將它作為一個完整的輕量電商作品來呈現，而不只是達成作業要求。

---

## 功能說明

**前台**
- 商品瀏覽
- 加入購物車
- 完整購買流程
- 購物車 icon 即時顯示數量（Redux Toolkit）
- 全域提示訊息（Redux Toolkit）

**後台**
- 商品管理（新增、編輯、刪除）
- 登入機制，控管操作權限
